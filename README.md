# tictactoeSockets
### Install Vuejs CLI
`npm install -g @vue/cli` <br>
Now, Use command:
`npm install` > 
`cd proj && npm install` > 
`npm run serve` <br>
Open another terminal and go into the proj_server director and enter the command: <br>
`npm install` <br>
`node server.js` <br>
Use the newtowk IP given in two browsers to run the multiplayer game. <br>
